Les rôles secrets
Chaque manche, tu reçois un rôle secret : Vérité ou Mensonge. Ne le montre à personne.
02
Répondre à voix haute
Une question s'affiche. Tout le monde répond à voix haute, dans l'ordre. Les menteurs doivent inventer. Les autres disent la vérité.
03
Débat libre
Après les réponses, échangez librement. Défendez-vous, accusez les autres, semez le doute.
04
Le vote
Chaque joueur vote pour le joueur qu'il pense être le menteur. Un seul vote par personne.
05
Les points
Menteur non démasqué → +2 pts. Joueur vérité qui vote juste → +1 pt.
